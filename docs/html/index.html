Trantor

Modular Object Relational Mapper (ORM) written in C++ 20 for SQLite databases

This is a documentation for the Trantor Project. To see high-level feature overview, head over to the Github.

Getting started

The best way to get started is to read the thorough guide to download, build, install and start using Trantor in your project. There is also a complete building documentation — there are packages for many platforms, including Windows, Linux and macOS. After that, there are various tutorials and examples and a complete feature guide explaining all aspects of the library.

Apart from that, various Trantor functionality is available through single-header libraries. Just download a file, @cpp #include @ce it in your project and you're ready to go! No buildsystem wrangling needed.

Contact & support

If you want to contribute to Trantor, if you spotted a bug, need a feature or have an awesome idea, you can get a copy of the sources from GitHub and start right away!
